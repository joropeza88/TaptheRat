# Tap the Rat

Mini juego casual inspirado en Whac-a-Mole, ambientado en una alacena donde un gato debe atrapar ratones que se asoman entre tablas de madera.

## Tecnologías

- Vue 3
- TypeScript
- Vite
- Tailwind CSS v4
- vite-plugin-pwa

## Juego

- Diseño orientado a móvil en vertical
- Sistema de niveles configurable
- Spawn dinámico de ratones
- Puntuación por objetivo
- Pantalla inicial, juego y victoria
- Preparado como PWA

## Estado actual

MVP funcional con:

- 1 nivel
- 3 tablas
- Ratones apareciendo arriba o abajo de cada tabla
- Animación del gato por sprite
- HUD superpuesto

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
